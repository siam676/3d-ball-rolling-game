# 3d-ball-rolling-game
its a 3d ball rolling game ..where basic features are included ( translate,rotation,scalling) ..with lighting , score board, cubes , coins , bridge , outside environment ,window,pictures frame and a fan as a continous rotating object.....

control panel 
A - LEFT
D - RIGHT
W - FORWARD
S - BACK
V - VIEWING CAMERA
L - LIGHT ON/OFF
Q/E - BALL SCALING
